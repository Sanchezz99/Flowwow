# Flowwow - Интернет-магазин цветов

## 🌸 О проекте

**Flowwow** — полноценный интернет-магазин цветов с бэкендом на **Django REST Framework**, базой данных **SQLite**, клиентскими страницами и админ-панелью для управления товарами, заказами и пользователями.

---

## 🚀 Демо-режим

После запуска сервер доступен по адресу:

| Раздел | URL |
|--------|-----|
| API | `http://127.0.0.1:8000/api/` |
| Админ-панель | `http://127.0.0.1:8000/admin/` |
| Тестовый эндпоинт | `http://127.0.0.1:8000/api/test/` |

**Учётные данные администратора:**
- Логин: `admin`
- Пароль: (задаётся при создании суперпользователя)

## 🛠️ Установка и запуск

### Требования

| Компонент | Версия |
|-----------|--------|
| Python | 3.10+ |
| pip | последняя |

### Инструкция

**1. Клонировать репозиторий**
```bash
git clone <repository-url>
cd flowwow
```
2. Создать виртуальное окружение

```bash
python -m venv venv
```
# Windows
```bash
venv\Scripts\activate
```
# Linux/Mac
```
source venv/bin/activate
```
3. Установить зависимости

```bash
pip install django djangorestframework djangorestframework-simplejwt django-cors-headers pillow
```
4. Применить миграции

```bash
python manage.py makemigrations
python manage.py migrate
```
5. Создать суперпользователя (админа)

```bash
python manage.py createsuperuser
```
6. Запустить сервер

```bash
python manage.py runserver
```
Сервер запустится на http://127.0.0.1:8000/

# 🧪 Тестовые данные
После применения миграций база данных пуста. Для тестирования рекомендуется:

Создать тестовых пользователей через /api/register/

Добавить товары через админ-панель

Загрузить изображения в папку media/flowers/

Пример тестового цветка
```json

{
  "name": "Красные розы",
  "description": "Классический букет из 51 красной розы",
  "price": 4990,
  "category": "romantic",
  "stock": 10,
  "rating": 4.8,
  "is_popular": true
}
```
# 🔐 Аутентификация
JWT для пользователей
JWT токен возвращается при входе и регистрации:

```json
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user_id": 1,
  "username": "user"
}
```
Использование в запросах:

```
Authorization: Bearer <token>
```
Админ-панель
Доступна по /admin/

Требуется логин и пароль суперпользователя
